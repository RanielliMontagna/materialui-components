# Material UI - Teste de componentes

## O que é o Material UI?

O Material UI é uma biblioteca de componentes React que implementa o Material Design.
O Material Design é um padrão de design desenvolvido pelo Google que visa a criação de interfaces de usuário mais bonitas e intuitivas.
Além disso, o MaterialUI hoje é a biblioteca de componentes mais utilizada no mercado.
Mais informações sobre o Material UI podem ser encontradas no site oficial:
[https://material-ui.com/](https://material-ui.com/)

## Como rodar o projeto

Para rodar o projeto, você deve ter o (Node.js)[https://nodejs.org/en/] instalado em sua máquina.

- Após clonar o projeto, você deve rodar o comando `npm install` ou `yarn` para instalar as dependências do projeto.

- Após instalar as dependências, você deve rodar o comando `npm start` ou `yarn start` para rodar o projeto.

- Pronto o projeto já está rodando em `http://localhost:3000`
